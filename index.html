metaknowledge is a Python3 library that simplifies bibliometric and computational sociology of knowledge research using Web of Science data. It reads a directory of plain text files containing meta-data on publications and citations, and writes to a variety of data structures that are suitable for quantitative, network, and text analyses. It handles large datasets (e.g. several million records) efficiently, and currently has functions for:

  • cleaning and preprocessing Web of Science data (e.g. identifying and deleting duplicate records)
  • extracting co-author, citation, and co-citation networks for published papers, authors, and journals
  • extracting multi-mode networks using any of the meta-data fields available in the Web of Science database (e.g. papers and keywords)
  • extracting data for textual analysis (e.g. titles, abstracts)

Datasets created with metaknowledge can be analyzed using the standard libraries for data analysis in Python, including NetworkX. It is also easy to write data to csv or graphml files for analysis and visualization in R, Visone, Gephi, or any other tools for data analysis.

metaknowledge also has a simple command line tool for extracting quantitative datasets and network files from Web of Science files. This makes the library more accessible to researchers who do not know Python, and makes it easier to quickly explore new datasets.
